OOP:Object Oriented Programming

Features of OOP
1)object : Any Entity which has own state and behaviour
ex: pen,paper,chair etc..

2)class : Collection of objects
ex: human body

3)abstraction : Hiding internal details and showing functionalities that process
is called abstraction
ex:Login Page

4)encapsualtion : Wrapping up of data or binding of data
ex:Capsule

5)inheritance : When One Object Acquire all the property 
and behaviour of parent class
ex:father-son

6)polymorphism : Many Ways to perform anything
ex: Multiple Ways
